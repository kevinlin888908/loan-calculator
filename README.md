# 貸款計算器 (Loan Calculator)

一個使用 C# 和 .NET 8 Windows Forms 開發的貸款本息定額每月攤還試算器。

## 功能特色

- 計算固定利率貸款的每月還款金額
- 支援不同的貸款金額、利率和年限設定
- 使用標準的貸款攤還公式進行精確計算
- 友好的中文介面

## 技術規格

- **.NET 8.0** - 最新的 .NET 框架
- **Windows Forms** - 桌面應用程式介面
- **C# 12.0** - 現代 C# 語言特性

## 預設設定

- 貸款金額：500,000 元
- 年利率：2%
- 貸款年數：20 年
- 每月攤還：約 25,120 元

## 數學公式

使用標準的貸款攤還公式：

```
PMT = PV × [r(1+r)^n] / [(1+r)^n - 1]
```

其中：
- PMT = 每月還款金額
- PV = 貸款本金
- r = 月利率 (年利率 ÷ 12)
- n = 總還款期數 (年數 × 12)

## 如何執行

```bash
dotnet run
```

## 系統需求

- Windows 10 或更新版本
- .NET 8.0 Runtime
