# ?? 行事曆管理系統 (Calendar Management System)

一個使用 C# 和 .NET 8 Windows Forms 開發的全功能行事曆管理應用程式。

## 功能特色

- ?? **行事曆顯示** - 直觀的月曆介面，點擊日期查看事件
- ? **事件管理** - 新增、編輯、刪除行事曆事件
- ?? **檔案輸出入** - 匯出事件到文字檔，支援從文字檔匯入事件
- ?? **智能過濾** - 按日期過濾事件，顯示每日事件統計
- ?? **快捷鍵支援** - 完整的鍵盤操作支援
- ?? **事件詳情** - 詳細的事件資訊顯示和編輯

## 技術規格

- **.NET 8.0** - 最新的 .NET 框架
- **Windows Forms** - 桌面應用程式介面
- **C# 12.0** - 現代 C# 語言特性
- **UTF-8 編碼** - 完整的中文支援

## 主要功能

### ?? 事件管理
- 新增事件：輸入標題、內容描述和日期時間
- 編輯事件：修改已存在的事件資訊
- 刪除事件：移除不需要的事件
- 事件驗證：確保事件資料的完整性

### ?? 檔案操作
- **匯出功能**：將所有事件匯出為格式化的文字檔
- **匯入功能**：從文字檔案載入事件資料
- **自動備份**：支援批次匯入和匯出操作
- **UTF-8 編碼**：確保中文字符正確處理

### ?? 快捷鍵
- `Ctrl+N`: 新增事件（焦點移至標題欄位）
- `Ctrl+E`: 匯出事件到文字檔
- `Ctrl+I`: 從文字檔匯入事件
- `F2`: 編輯選中的事件
- `Delete`: 刪除選中的事件
- `ESC`: 取消編輯模式

## 使用範例

### 新增事件
1. 在行事曆上選擇日期
2. 輸入事件標題（例如：「房東簽約」）
3. 輸入事件內容（例如：「房東一月三號下午三點要簽約」）
4. 設定準確的日期時間
5. 點擊「新增事件」按鈕

### 匯出事件
1. 使用功能表：檔案 → 匯出事件到文字檔
2. 或按快捷鍵 `Ctrl+E`
3. 選擇儲存位置和檔案名稱
4. 事件將以結構化格式儲存

### 匯入事件
1. 使用功能表：檔案 → 從文字檔匯入事件  
2. 或按快捷鍵 `Ctrl+I`
3. 選擇要匯入的文字檔
4. 確認匯入的事件清單

## 文字檔格式

匯出的文字檔包含以下資訊：
```
=== 行事曆事件列表 ===
匯出時間：2024年12月24日 10:30:00
總事件數：3

事件編號：1
標題：房東簽約
時間：2024年01月03日 星期三 15:00
內容：房東一月三號下午三點要簽約
----------------------------------------
```
